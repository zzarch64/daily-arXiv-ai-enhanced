### [1] [ChipCraftBrain: Validation-First RTL Generation via Multi-Agent Orchestration](https://arxiv.org/abs/2604.19856)
*Cagri Eryilmaz*

Main category: cs.AR

TL;DR: ChipCraftBrain是一个结合符号-神经推理与自适应多智能体编排的框架，用于自动化RTL代码生成。在VerilogEval-Human上达到97.2% pass@1，在CVDP工业基准上达到94.7%，并通过层次化解构成功完成RISC-V SoC案例（8/8模块通过lint检查并FPGA验证），相比单次生成提升36-60个百分点，且API成本远低于同类方法。


<details>
  <summary>Details</summary>
Motivation: 现有LLM生成RTL代码的单次生成仅60-65%功能正确率；多智能体方法如MAGE虽达95.9%但未在工业级基准（如NVIDIA CVDP）上测试，缺乏综合感知且API成本高。

Method: (1) 基于PPO策略（168维状态）的自适应多智能体编排（6个专用智能体），另设MPC规划器作为替代方案；(2) 混合符号-神经架构：算法性求解K-map和真值表，专用智能体处理波形时序与通用RTL；(3) 知识增强生成：321个模式库+971个开源参考实现，焦点感知检索；(4) 层次化规范分解：依赖排序的子模块+接口同步。

Result: VerilogEval-Human: 97.2% pass@1（7次运行范围96.15-98.72%，最佳154/156）；CVDP 302问题子集: 94.7% pass@1（286/302，3次平均），每类别比单次基线提升36-60个百分点；ACE-RTL四个共享类别中领先三个，且每次问题尝试次数少约30倍；RISC-V SoC: 8/8 lint-passing模块（689 LOC）通过FPGA验证，单次生成完全失败。

Conclusion: ChipCraftBrain通过符号-神经混合推理与自适应多智能体编排，显著提升了RTL生成的正确率、工业级泛化能力和综合感知能力，同时大幅降低了API成本。层次化解构方法使复杂SoC级设计成为可能，在多个基准上达到SOTA。

Abstract: Large Language Models (LLMs) show promise for generating Register-Transfer Level (RTL) code from natural language specifications, but single-shot generation achieves only 60-65% functional correctness on standard benchmarks. Multi-agent approaches such as MAGE reach 95.9% on VerilogEval yet remain untested on harder industrial benchmarks such as NVIDIA's CVDP, lack synthesis awareness, and incur high API costs.
  We present ChipCraftBrain, a framework combining symbolic-neural reasoning with adaptive multi-agent orchestration for automated RTL generation. Four innovations drive the system: (1) adaptive orchestration over six specialized agents via a PPO policy over a 168-dim state (an alternative world-model MPC planner is also evaluated); (2) a hybrid symbolic-neural architecture that solves K-map and truth-table problems algorithmically while specialized agents handle waveform timing and general RTL; (3) knowledge-augmented generation from a 321-pattern base plus 971 open-source reference implementations with focus-aware retrieval; and (4) hierarchical specification decomposition into dependency-ordered sub-modules with interface synchronization.
  On VerilogEval-Human, ChipCraftBrain achieves 97.2% mean pass@1 (range 96.15-98.72% across 7 runs, best 154/156), on par with ChipAgents (97.4%, self-reported) and ahead of MAGE (95.9%). On a 302-problem non-agentic subset of CVDP spanning five task categories, we reach 94.7% mean pass@1 (286/302, averaged over 3 runs), a 36-60 percentage-point lift per category over the published single-shot baseline; we additionally lead three of four categories shared with NVIDIA's ACE-RTL despite using roughly 30x fewer per-problem attempts. A RISC-V SoC case study demonstrates hierarchical decomposition generating 8/8 lint-passing modules (689 LOC) validated on FPGA, where monolithic generation fails entirely.

### [2] [Efficient Page Migration in Hybrid Memory Systems](https://arxiv.org/abs/2604.19932)
*Upasna,Venkata Kalyan Tavva*

Main category: cs.AR

TL;DR: 本文提出Duon，一种通过将更新后的页映射信息直接存储在TLB和页表中来消除异构内存架构中页迁移导致的TLB shootdown和缓存行失效开销的方法，在IPC上比现有技术提升3.87%。


<details>
  <summary>Details</summary>
Motivation: 在异构内存架构（HMA）的Flat Address Space组织方式下，频繁访问的数据页需要从慢速内存重映射到快速内存（如HBM），但页迁移过程中的TLB shootdown和缓存行失效带来了严重的性能开销。

Method: 提出Duon方案，利用扩展的TLB和页表结构，将页迁移后的更新映射信息直接存储在TLB和页表本身中，从而避免了TLB shootdown和缓存行失效操作。

Result: 相比现有最优技术，IPC提升了3.87%，且该方法兼容现有的各种页迁移策略。

Conclusion: Duon有效消除了页迁移的开销，提升了异构内存系统的整体性能和效率，具有较强的通用性。

Abstract: Heterogeneous Memory Architecture (HMA) aims to optimize memory usage by leveraging a combination of memory types, such as high-bandwidth memory (HBM), commodity DRAM, and non-volatile memory (NVM), when utilized as main memory. To achieve maximum performance benefits, frequently accessed data pages are prioritized for storage in the faster HBM, while less frequently accessed pages are stored in slower memory types like DRAM or NVM. This enables a more efficient allocation of memory resources and improves overall system performance. In a Flat Address Space memory organization, all memory types, both fast and slow, are treated as a unified memory pool. This approach increases the overall memory capacity accessible to the system. In Flat Address Space organization, frequently accessed data pages may need to be remapped from slower memory to faster memory to improve memory access times. Such relocation requires changes to the data/states in the TLB (TLB shootdown) and the processor cache (cache line invalidations), leading to performance degradation. To address these inefficiencies, we propose a novel solution called Duon. The goal of Duon is to eliminate the overheads associated with page migration in systems using Extended TLB and Page Table. Specifically, our approach ensures that the updated mapping information for remapped pages is carefully stored directly in the TLB and page table itself. By doing so, the need for TLB shootdown and cache line invalidation after page migration is eliminated. Consequently, our proposal results in an overall improvement in IPC by 3.87% over existing state-of-the-art techniques, enhancing the efficiency and performance of heterogeneous memory systems. Further, our approach can work with any of the existing page migration policies and improve the performance.

### [3] [Algorithm and Hardware Co-Design for Efficient Complex-Valued Uncertainty Estimation](https://arxiv.org/abs/2604.19993)
*Zehuan Zhang,Mark Chen,He Li,Wayne Luk*

Main category: cs.AR

TL;DR: 本文首次提出基于Dropout的贝叶斯复数神经网络（BayesCVNN），实现复数任务的量化不确定性，并通过自动化搜索与FPGA加速器生成框架，在性能和硬件效率上均优于现有方法。


<details>
  <summary>Details</summary>
Motivation: 现有复数神经网络（CVNNs）无法量化预测不确定性，限制了其在安全攸关场景的应用。

Method: 1) 提出Dropout-based贝叶斯CVNN实现不确定性量化；2) 引入自动化搜索方法，利用复数双层特性（层混合与部分混合）优化配置；3) 提出FPGA加速器生成框架，包含优化构建模块。

Result: 自动搜索找到的最优配置在性能与硬件成本上均优于手工设计；FPGA加速器相比GPU实现获得约4.5倍和13倍加速，功耗不到GPU的10%。

Conclusion: BayesCVNN首次将不确定性量化引入复数神经网络，结合自动化搜索与FPGA部署，在算法与硬件两方面均实现显著提升，具有广泛适用性和硬件实现高效性。

Abstract: Complex-Valued Neural Networks (CVNNs) have significant advantages in handling tasks that involve complex numbers. However, existing CVNNs are unable to quantify predictive uncertainty. We propose, for the first time, dropout-based Bayesian Complex-Valued Neural Networks (BayesCVNNs) to enable uncertainty quantification for complex-valued applications, exhibiting broad applicability and efficiency for hardware implementation due to modularity. Furthermore, as the dual-part nature of complex values significantly broadens the design space and enables novel configurations based on layer-mixing and part-mixing, we introduce an automated search approach to effectively identify optimal configurations for both real and imaginary components. To facilitate deployment, we present a framework that generates customized FPGA-based accelerators for BayesCVNNs, leveraging a set of optimized building blocks. Experiments demonstrate the best configuration can be effectively found via the automated search, attaining higher performance with lower hardware costs compared with manually crafted models. The optimized accelerators achieve approximately 4.5x and 13x speedups on different models with less than 10% power consumption compared to GPU implementations, and outperform existing work in both algorithm and hardware aspects. Our code is publicly available at: https://github.com/zehuanzhang/BayesCVNN.git.

### [4] [EnergAIzer: Fast and Accurate GPU Power Estimation Framework for AI Workloads](https://arxiv.org/abs/2604.20105)
*Kyungmi Lee,Zhiye Song,Eun Kyung Lee,Xin Zhang,Tamar Eilam,Anantha P. Chandrakasan*

Main category: cs.AR

TL;DR: EnergAIzer是一种轻量级GPU功耗预测框架，通过分析AI工作负载中内核的结构化优化模式来快速预测硬件利用率，从而绕过昂贵的仿真或硬件分析，在数秒内实现与耗时方法相当的功耗预测精度（Ampere GPU误差8%，H100预测误差7%）。


<details>
  <summary>Details</summary>
Motivation: 现有GPU功耗模型存在可扩展性瓶颈：传统方法依赖昂贵的周期级仿真或硬件性能分析，获取硬件利用率输入耗时过长（数小时），无法满足AI工作负载快速增长的功耗管理需求。

Method: 核心洞察是AI工作负载中的内核常采用结构化优化模式，可解析性地确定内存流量和执行时间线。EnergAIzer利用这些模式构建性能模型作为经验数据拟合的分析框架，自然暴露模块级利用率，再将预测的利用率输入功耗模型估算动态功耗。

Result: 在NVIDIA Ampere GPU上功耗误差仅8%，与传统的周期级仿真或硬件分析方法相当；预测NVIDIA H100功耗误差仅7%。将估计耗时从数小时缩短至数秒。

Conclusion: EnergAIzer为AI工作负载提供了快速且准确的功耗预测，使得功耗感知的设计空间探索（如频率缩放和架构配置）变得切实可行。

Abstract: As AI workloads drive increases in datacenter power consumption, accurate GPU power estimation is critical for proactive power management. However, existing power models face a scalability bottleneck not in the modeling techniques themselves, but in obtaining the hardware utilization inputs they require. Conventional approaches rely on either costly simulation or hardware profiling, which makes them impractical when rapid predictions are required.
  This work presents EnergAIzer, which addresses this scalability bottleneck by developing a lightweight solution to predict utilization inputs, reducing the estimation walltime from hours to seconds. Our key insight is that kernels in AI workloads commonly employ optimizations that create structured patterns, which analytically determine memory traffic and execution timeline. We construct a performance model using these patterns as an analytical scaffold for empirical data fitting, which also naturally exposes module-level utilization. This predicted utilization is then fed into our power model to estimate dynamic power consumption.
  EnergAIzer achieves 8% power errors on NVIDIA Ampere GPUs, competitive with traditional power models with elaborate cycle-level simulation or hardware profiling. We demonstrate EnergAIzer's exploration capabilities for frequency scaling and architectural configurations, including forecasting the power of NVIDIA H100 with just 7% error. In summary, EnergAIzer provides fast and accurate power prediction for AI workloads, paving the way for power-aware design explorations.

### [5] [A Novel Low-Power Cache Architecture Based on 6-Transistor SRAM Cells](https://arxiv.org/abs/2604.20176)
*Naser Khatti Dizabadi,Ceyda Elcin Kaya*

Main category: cs.AR

TL;DR: 本文提出了一种基于6T SRAM单元串联的低功耗缓存架构，通过列内相邻单元的串联拓扑利用堆叠效应抑制漏电流，在保持标准6T单元结构的前提下降低了缓存静态功耗。


<details>
  <summary>Details</summary>
Motivation: 降低SRAM缓存中因漏电流导致的静态功耗，同时不增加存储单元本身的晶体管数量，以维持面积效率。

Method: 将列内相邻的6T SRAM单元重新配置为串联拓扑结构，利用堆叠效应在保持（hold）操作时抑制漏电流；同时对缓存的寻址和读出结构（列组织与读取路径）进行了相应调整。

Result: 基于Keysight ADS的瞬态仿真表明，与传统SRAM互联方案相比，所提架构在维持标准6T SRAM单元的情况下有效降低了漏电功耗。

Conclusion: 通过串联单元拓扑实现堆叠效应是一种在不增加单元晶体管数前提下降低缓存漏电功耗的有效方法，但需配合寻址与读出电路的修改。

Abstract: This paper presents a low-power cache architecture based on the series interconnection of conventional 6-transistor static random-access memory (6T SRAM) cells. The proposed approach aims to reduce leakage power in SRAM-based cache memories without increasing the transistor count of the memory cell itself. In the proposed architecture, adjacent cells within a column are reconfigured in a serial topology, thereby exploiting the stacking effect to suppress leakage current, particularly during hold operation. This architectural modification requires corresponding changes to the addressing and sensing structure of the cache, including adjustments to the column organization and readout path. To evaluate the proposed method, transient simulations were carried out using Keysight ADS. The simulation results show that the proposed architecture reduces leakage power compared with the conventional SRAM interconnection scheme while preserving the use of standard 6T SRAM cells.

### [6] [Evaluating Computing Platforms for Sustainability: A Comparative Analysis of FPGAs against ASICs, GPUs, and CPUs](https://arxiv.org/abs/2604.20638)
*Chetan Choppali Sudarshan,Aman Arora,Vidya A Chhabria*

Main category: cs.AR

TL;DR: GreenFPGA是一个估算FPGA全生命周期碳足迹的工具，考虑设计、制造、可重构性（复用）、运行、废弃、测试和回收等环节的不确定性，发现FPGA在频繁变更、多样化工作负载和低容量应用场景下比ASIC、GPU和CPU更可持续。


<details>
  <summary>Details</summary>
Motivation: 气候变化问题强调可持续计算的必要性，而现有工具缺乏对FPGA全生命周期碳足迹（包括操作和隐含碳足迹）的综合估算，尤其是考虑到其可重构性带来的碳足迹分摊优势。

Method: 提出GreenFPGA工具，建模FPGA从设计、制造、可重构复用、运行到废弃、测试和回收的全生命周期碳足迹，并考虑建模中的不确定性；在假设性能相同（iso-performance）的条件下，与ASIC、GPU、CPU进行对比分析。

Result: 实验表明，应用类型、寿命、使用时间和产量等参数显著影响总碳足迹；在成对性能相同的案例研究中，FPGA在频繁更换、多样化工作负载和低容量应用中比ASIC、GPU、CPU更可持续。

Conclusion: GreenFPGA能够识别出FPGA比ASIC、GPU、CPU更可持续的部署场景，强调了跨平台分析CFP的重要性，为可持续计算提供决策支持。

Abstract: Climate change concerns emphasize the need for sustainable computing. Modeling the carbon footprint (CFP), including operational and embodied CFP from semiconductor use, manufacture and design, is essential. Field programmable gate arrays (FPGAs) stand out as promising platforms due to their reconfigurability across various applications, enabling the amortization of embodied CFP across multiple applications. This paper introduces GreenFPGA, a tool estimating the total CFP of FPGAs over their lifespan, considering uncertainties in CFP modeling. It accounts for CFP during design, manufacturing, reconfigurability (reuse), operation, disposal, testing, and recycling. GreenFPGA identifies deployment regimes in which FPGAs can be more sustainable than ASICs, GPUs, and CPUs under the modeled iso-performance assumptions. Experimental results highlight the importance of analyzing applications across different computing platforms to assess their CFP while varying parameters such as application type, lifetime, usage time, and volume impact their total CFP. Across the evaluated pairwise iso-performance case studies with ASICs, GPUs, and CPUs, FPGAs can be more sustainable under specific deployment regimes involving frequently changing, diverse workloads and low-volume applications.
